# 贡献指南：壮大女娲智库

太平世界，同此凉热。如果你也想为本系统添砖加瓦，请遵循以下原则。

## 核心原则：灵魂重于形式

我们提炼的不是角色的语录，而是角色的**思维骨架**。当你贡献新的案例或启发式模型时，请自问：

- 这是否触及了事物的主要矛盾？
- 这是否具备跨越时空的生命力？
- 这是否在泥土中经过了实地的检验？

## 表达禁令：拒绝 AI 数字化

本项目的核心魅力在于“去 AI 味”。在提交 Pull Request 时，请务必遵守以下审美规范：

- **严禁 Markdown 加粗**：杜绝机械的强调。
- **严禁数字列表**：不要使用 1. 2. 3. 这种机器人排版。请使用自然段落、空行或无序的感性小标题。
- **严禁官话套话**：不说“这是一个很好的切入点”。直接说“我看你这个问题，根子在这个地方”。

## 提交建议

- **案例研究**：我们非常欢迎针对现实具体困境（如职场纠纷、技术选型、人生重大抉择）的对话范本。
- **思维工具**：如果你发现导师在历史文献中还有某种未被提炼的底层算法，请直接提交到 `references/`。

愿你在泥土中找到自己的根据地，愿我们一同完成这场历史思维的万里长征。
